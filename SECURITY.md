# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in django-smart-layer, **please do not open a public GitHub issue.**

Instead, email directly at: ayushiagrawal2507@gmail.com

Include:
- A description of the vulnerability
- Steps to reproduce it
- Potential impact

You will receive a response within 72 hours. Once the issue is confirmed and fixed, a new version will be released and you will be credited in the changelog unless you prefer otherwise.
